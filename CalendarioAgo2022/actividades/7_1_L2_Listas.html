Laboratorio: Listas o arreglos
.
.

   Descripción:  
    • Desarrolla, en conjunto con el profesor, algunos problemas que implican el uso de listas o arreglos.

 Objetivos:  
    • Aplicar el concepto de listas o arreglos en la solución de problemas.
    • Aplicar los conocimientos sobre funciones.

   Modalidad:  
    • Sesión plenaria.
 
Instrucciones:
  

    • Implementa las siguientes funciones haciendo uso de listas o arreglos:

 
  Ejercicios: 
    • La función crea_lista (tam) que recibe el tamaño de la lista o arrreglo. La función deberá pedir un valor numérico al usuario para cada casilla de la lista y al final devolver la lista creada. 

Casos de prueba: Si la lista a crear es de tamaño 3, entonces se pedirán tres valores. 

  • Input: Tamaño de la lista
  • Input: Valor 1
  • Input: Valor 2
  • Input: Valor 3
  • Output: La lista creada con los valores dados por el usuario

Input: 3
Input: 48
Input: 4
Input: 7
Output: [48, 4, 7]

Input: 2
Input: 3
Input: 5
Output: [3, 5]

Input: 0
Output: [ ]

    • La función inicializa_lista (lista) que recibe una lista. La función deberá cambiar cada localidad de la lista con un número aleatorio entre -25 y 25. La función no crea una nueva lista, sino que modifica la que recibe como parámetro. Para mandar llamar esta función antes debes usar crea_lista para asegurarte que tienes valores que cambiar. 

Casos de prueba: Si la lista a crear es de tamaño 3, entonces se pedirán tres valores. 

  • Input: Tamaño de la lista en crea_lista (tam)
  • Input: Valor 1
  • Input: Valor 2
  • Input: Valor 3
  • Output: La lista modificada con números aleatorios entre -25 y 25.

 Output:

[5, -25, 8]

    • La función lista_positiva (lista) que recibe un arreglo o lista de números. La función deberá hacer positivas todas aquellas localidades que tengan un valor negativo. Para mandar llamar esta función antes debes usar crea_lista para asegurarte que tienes valores en la lista.

Ejemplo: Si el arreglo original tiene asignados los siguientes valores:

-1  3  5  10  -3  12  -5 
0  1  2  3  4  5  6 

Al aplicar la función lista_positiva (lista) el arreglo o lista quedaría con los siguientes valores:

1  3  5  10  3  12  5 
0  1  2  3  4  5  6 

Casos de prueba: Si la lista a crear es de tamaño 3, entonces se pedirán tres valores. 

  • Input: Tamaño a usar en crea_lista (tam)
  • Input: Valor 1
  • Input: Valor 2
  • Input: Valor 3
  • Output: La lista modificada con solamente valores positivos

[-20, 4, -7]

Input: -20
Input: 4
Input: -7
Output:

[20, 4, 7]

    • La función media (lista) que recibe un arreglo o lista de números y devuelve el cálculo de la media de los elementos en la lista. Para mandar llamar esta función antes debes usar crea_lista para asegurarte que tienes valores en la lista.

Ejemplo: Suponiendo que la lista tiene asignados los siguientes valores:  

2  4  6  8 
0  1  2  3 

El resultado de la función sería: 5         

               

Casos de prueba: Si la lista a crear es de tamaño 3, entonces se pedirán tres valores. 

  • Input: Tamaño a usar en crea_lista (tam)
  • Input: Valor 1
  • Input: Valor 2
  • Input: Valor 3
  • Output: La media poblacional de los valores de la lista (número decimal con dos decimales)

[48, 4, 7]

Input: 3
Input: 48
Input: 4
Input: 7
Output: 19.67

[3, 8]

Input: 2
Input: 3
Input: 8
Output: 5.50

[ ]

Input: 0
Output: 0.00
